## 截图建议

### 获取截图的 3 种方式
1. **真机截图**（推荐）：在手机上安装APP，逐页面截图
2. **模拟器截图**：在 HBuilderX 运行到 Android 模拟器，逐页面截图
3. **H5模式截图**：运行 `npm run dev:h5`，浏览器开发者工具手机模式截图

### 截图注意事项
- 截图清晰度要高，文字可辨认
- 截图中软件名称需与申请名称一致（云客CRM / CRM外呼助手）
- 不要出现其他公司的商标或名称
- 建议使用竖屏手机截图（与实际使用场景一致）
- 尺寸无硬性要求，脚本会自动缩放适配

## 生成 PDF 命令

截图全部放好后，在终端执行：

```powershell
cd "D:\kaifa\CRM - 1.8.0\crmAPP"
node scripts/generate-doc-pdf.js
```

### 输出文件
- **HTML**: `crmAPP/docs/document-identification.html`
- **PDF**: `crmAPP/docs/文档鉴别材料.pdf`

> 没放截图的位置会显示虚线占位框，不影响生成。可以先放部分截图预览，后续补全再重新运行。
