第一章 软件概述

1.1 软件简介

云客CRM外呼助手软件是一款基于uni-app框架开发的移动端电销外呼助手应用，配合云客CRM客户关系管理系统使用，专为电话销售团队设计。采用Vue 3 + TypeScript技术栈，支持Android和iOS双平台，通过WebSocket实现PC端远程控制手机拨号。

软件具备通话录音自动采集、数据实时同步统计、扫码快速绑定设备等特色功能。

1.2 软件用途

本软件主要用于以下场景：

  • 配合PC端CRM系统远程控制手机拨号
  • 手机端主动外呼，拨号盘手动输入号码
  • 自动采集通话录音同步服务器
  • 实时统计通话数据，辅助管理决策
  • 通话结束后记录跟进信息和意向标签

1.3 运行环境

项目要求
操作系统Android 7.0+ / iOS 12.0+
设备智能手机
网络WiFi或移动数据
配套云客CRM后端服务

第四章 技术架构说明

4.1 前后端交互方式

采用前后端分离架构。RESTful API用于登录认证、数据查询提交，请求头携带JWT令牌。WebSocket用于实时通信：接收拨号指令、上报状态、系统通知。

PC端 --> [WebSocket] --> 服务器 --> [WebSocket] --> APP APP --> [HTTP API] --> 服务器 --> [数据库] --> 持久化

4.2 WebSocket通信协议

采用事件驱动模式，核心事件：

事件方向说明
dial_command服务器→APPPC端拨号指令
call_state_changeAPP→服务器通话状态变化
device_bind双向设备绑定状态
heartbeat双向心跳保活

实现自动重连机制，断开后按递增间隔重连，最长30秒。

4.3 多租户架构

支持多租户SaaS部署，不同企业部署独立服务实例。APP启动时配置目标服务器，支持域名和IP两种格式。

4.4 数据安全

多层防护措施：

  • 传输加密：HTTPS/WSS协议
  • 身份认证：JWT令牌机制
  • 本地加密：AES-256加密存储
  • 录音安全：上传后删除本地
  • 权限控制：最小权限原则
